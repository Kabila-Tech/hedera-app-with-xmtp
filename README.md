# Hedera dApp with HIP820 & EIP155 support

## Setup

1. Set VITE_REOWN_PROJECT_ID variable in `.env.local`:

```sh
cp .env.local.example .env.local
```

2. Install dependcies and run example

** with yarn **

```sh
yarn install
yarn dev
```

** with npm **

```sh
npm install --force
npm run dev
```
